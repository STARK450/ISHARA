# ISHARA — Online Bags Store

ISHARA is a modern online bags storefront built using TypeScript and Vite.  
This project demonstrates a clean, responsive user interface, a structured product catalog, and core frontend e-commerce concepts. It is suitable for portfolio presentation and can be extended into a full-scale shopping application.
---

## Features

- Responsive product listing for browsing bags
- Built with modern frontend technologies
- Global state management using React Context
- Component-based and reusable architecture
- Easy to extend with cart, filters, routing, and backend integration

---

## Tech Stack

Framework: React  
Language: TypeScript  
Bundler: Vite  
State Management: React Context API  
Styling: CSS / Tailwind CSS  
Deployment: Vercel

---
